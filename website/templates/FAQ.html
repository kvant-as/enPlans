Как начать работу с системой?

Начать работу с EnPlans просто даже для новичков. Следуйте этому пошаговому руководству, и вы сможете создать свой первый план за 15 минут.

1

Регистрация

Укажите электронную почту, придумайте пароль и подтвердите его. На указанную почту придет код подтверждения.

2

Подтверждение Электронного адреса

Введите код из письма

3

Ввод персональных данных

Заполните личные данные: ФИО, номер телефона и должность.

4

Выбор типа пользователя

Выберите один из вариантов:

  • Респондент — создает и заполняет планы
  • Аудитор — проверяет планы
  • Утверждающая организация — утверждает планы
4

Поиск и выбор организации

Используйте поиск, чтобы найти свою организацию по названию, УНП, или ОКПО. Затем нажмите "Зарегистрироваться".

5

Переход в профиль

После выбора организации и нажатия кнопки "Зарегистрироваться" вы будете автоматически перенаправлены на страницу профиля.

Чеклист для начала работы:

Проверьте готовность к работе

Выполните все пункты для полноценной работы в системе

Создание плана энергоэффективности

Создание плана энергоэффективности в EnPlans — это простой пошаговый процесс, который занимает всего несколько минут. Система проведет вас через все необходимые этапы.

1

Переход к созданию плана

В главном меню выберите "Создать" или нажмите соответствующую кнопку на панели управления на странице Планы.

2

Заполнение базовых параметров

Укажите обязательные параметры: год, показатель энергосбережения, задания по экономии ТЭР, долю местных ТЭР и ВИЭ.

3

Открытие плана

После ввода обязательных параметров вы будете перенаправлены на страницу с планами. Нажмите «Просмотр» для открытия плана и перехода к заполнению данных.

4

Выбор типа плана

Определите категорию организации по объему энергопотребления:

📋
До 25 тыс. т. у. т.

Упрощенный формат отчетности для организаций с меньшим потреблением

📊
Более 25 тыс. т. у. т.

Расширенный формат с детальными расчетами для крупных потребителей

5

Заполнение данных

Внесите информацию о энергопотреблении, заполнив таблицы мероприятий и показателей.

Важно

Все изменения фиксируются в системе автоматически после нажатия кнопок Применить изменения или Добавить.

✓ Контроль плана с последующей отправкой на проверку

Согласование планов энергоэффективности

Согласование и утверждение планов в EnPlans выполняется специально назначенными аудиторами. Процесс проверки включает несколько этапов.

1

Получение плана

Изначальный статус плана "Не просмотрен".

2

Проверка данных

Последовательная проверка данных:

  • основные показатели (энергосбережение, доля местных ТЭР и ВИЭ)
  • мероприятия по экономии и увеличению ТЭР
  • показатели ТЭР
3

Вынесение вердикта

По результатам проверки аудитор выбирает один из вариантов:

Есть ошибки
Указываются конкретные ошибки, план возвращается на доработку
Одобрить
План соответствует требованиям, переводится в статус "Согласован"
Отменить
План переходит в изначальное состояние "Не просмотренный"

Что происходит после проверки?

✗

Если есть ошибки

План возвращается респонденту с комментариями. Статус меняется на "Есть ошибки"

✓

Если Согласовано

План переводится в статус "Согласован" и считается официально принятым

↩️

Если отменено

Статус плана возвращается к изначальному "Не просмотренный"

Важно

Все изменения статуса плана автоматически фиксируются в системе и отображаются в истории изменений.

Экспорт плана

EnPlans предоставляет несколько форматов экспорта для различных целей: от аналитики в Excel до официальных проектов плана для подачи.

1

Переход к экспорту

В меню плана выберите Экспорт.

2

Настройка параметров экспорта

Выберите формат файла xlsx, pdf, xml, используйте фильтры по статусу и году, отметьте нужные планы.

3

Формирование и скачивание

Нажмите кнопку Экспортировать. Система автоматически сформирует архив с выбранным форматом файла/файлов и предложит его скачать.

📈

Формат (.xlsx)

Для анализа и работы с данными. Полные таблицы планов и мероприятий.

📄

Формат (.pdf)

Для официальной подачи. Форматированный проект плана, готов к печати.

⚙️

Формат (.xml)

Для интеграции с программами. Структурированные данные, машинно-читаемый формат.

Как пройти проверку ЭЦП

Для отправки планов в системе EnPlans требуется использование электронной цифровой подписи (ЭЦП).

Важное требование

Для осуществления функции "отправить план" должно быть установлено ПО "Менеджер сертификатов" от Авест с настроенным сертификатом ЭЦП.

Виды сертификатов

✅

Действительный сертификат

Срок действия сертификата является удовлетворительным

❌

Недействительный сертификат

Срок действия сертификата истек, требуется обновление

Где находится файл сертифаката?

1

Откройте ПО "Менеджер сертификатов"

Запустите программу "Менеджер сертификатов" от Авест.

2

Выберите действительный сертификат

Выберите "Действительный" сертификат из списка.

3

Экспортируйте сертификат в файл

Кликните правой кнопкой по сертификату и выберите "Экспорт сертификата в файл".

4

Импортируйте сертификат в систему

При отправке плана используйте полученный файл сертификата.

Как рассчитывается разница в таблице "Показатели"?

В таблице «Показатели ТЭР» столбец «Изменение ТЭР прогнозного года к предыдущему (увеличение +, снижение -)» рассчитывается автоматически на основе введенных данных.

Как это работает?

1

Ввод значений

Пользователь вводит значения для каждого показателя в его единицах измерения (тонны, тыс. м³, Гкал, тыс. кВт·ч и т.д.) в полях прогнозного и предыдущего года.

2

Пересчет в т у.т.

Система автоматически пересчитывает введенные значения в тонны условного топлива (т у.т.) с учетом коэффициента пересчета:

Значение в т у.т. = Значение в ед.изм. × Коэффициент пересчета

Коэффициент пересчета может быть скорректирован для каждого года отдельно.

3

Расчет разницы

Разница вычисляется как разность между значениями прогнозного и предыдущего года в т у.т.:

Разница = Прогнозный год − Предыдущий год
  • Положительная разница → увеличение (+)
  • Отрицательная разница → снижение (-)

Цветовая индикация

Ячейка с разницей автоматически окрашивается в зависимости от типа показателя:

Для местных видов топлива и включая:

  • виды КПТ, которые включаются в строку «из него местные виды топлива и отходы»
  • «из него местные виды топлива и отходы»
  • «из них возобновляемые»
  • «Целевой показатель по доле местных ТЭР в КПТ»
  • «Целевой показатель по доле возобновляемых источников энергии в КПТ»
  • «энергия воды, ветра, солнца, геотермальных источников»
↓ Снижение — красный
↑ Увеличение — зеленый

Для не местных видов топлива и включая:

  • виды КПТ, которые не включаются в строку «из него местные виды топлива и отходы»
  • «Котельно-печное топливо (КПТ), израсходовано всего, в том числе:»
  • «Электроэнергия, израсходовано всего»
  • «Электроэнергия, выработанная собственными энергоисточниками, в том числе:»
  • «Теплоэнергия, израсходовано всего»
  • «Теплоэнергия, произведенная собственными энергоисточниками, в том числе:»
  • «Суммарное потребление ТЭР»
↓ Снижение — зеленый
↑ Увеличение — красный

Пример расчета

Показатель Ед. изм. Пред. год (ед.) Пред. год (т у.т.) Прогноз (ед.) Прогноз (т у.т.) Разница Тип
Газ природный тыс. м³ 100 115,0 95 109,3 ↓ -5,7 Местный
Газ природный тыс. м³ 100 115,0 120 138,0 ↑ +23,0 Местный
Дрова пл. м³ 50 11,4 80 18,2 ↑ +6,8 Неместный
Дрова пл. м³ 50 11,4 30 6,8 ↓ -4,6 Неместный

Важно

Разница пересчитывается автоматически при любом изменении значений. Для корректного расчета убедитесь, что коэффициенты пересчета установлены правильно.

Заполнение плана энергоэффективности

После создания плана и выбора типа (до 25 тыс. т у.т. или свыше 25 тыс. т у.т.) необходимо заполнить три основных раздела:

Раздел 1. Показатели использования ТЭР

В этом разделе отражаются объемы потребления топливно-энергетических ресурсов за отчетный, оценочный и прогнозный периоды.

1

Добавление показателей

При создании плана таблица показателей пустая. Для добавления показателя нажмите кнопку «Добавить» в левом верхнем углу страницы.

2

Выбор показателя из справочника

В открывшемся окне выберите необходимый показатель кликнув левой кнопкой мыши.

3

Ввод значений

Для каждого показателя необходимо ввести значения. Пользователь может самостоятельно выбирать, какие года отображать в столбцах, и вводить значения в тоннах условного топлива (т у.т.).

Как это работает:

  • Пользователь вводит значение в т у.т. в поле прогнозного года
  • Система автоматически пересчитывает его в натуральные единицы измерения (тонны, тыс. м³, Гкал и т.д.) с учетом коэффициента пересчета
  • Коэффициент пересчета можно редактировать вручную для каждого года отдельно
  • Если единица измерения показателя — т у.т., редактирование коэффициента заблокировано
Ввод в
т у.т.
↓

Для показателей «прочие отходы» и «прочие виды топлива» заполнение отличается:

  • Необходимо дополнительно ввести наименование конкретного вида отхода или топлива
  • Выбрать категорию из выпадающего списка:
Импортируемые виды топлива
Местные виды топлива и отходы
Возобновляемые источники энергии

От выбранной категории зависит, будет ли показатель включен в строки «из него местные виды топлива и отходы» и «из них возобновляемые».

4

Редактирование и удаление показателей

Для редактирования показателя нажмите кнопку «✎» в строке показателя или в меню над таблицей. Для удаления — кнопку «✕».

Обратите внимание

Некоторые показатели не могут быть удалены:

  • Котельно-печное топливо (КПТ) всего
  • из него местные виды топлива и отходы
  • из них возобновляемые
  • Суммарное потребление ТЭР
  • Электроэнергия всего
  • Теплоэнергия всего
  • Целевой показатель энергосбережения
  • Доля местных ТЭР в КПТ
  • Доля ВИЭ в КПТ

Раздел 2. Мероприятия по экономии ТЭР

В этом разделе отражаются мероприятия, направленные на сокращение потребления топливно-энергетических ресурсов.

1

Добавление мероприятия

Нажмите кнопку «Добавить» в таблице мероприятий по экономии ТЭР.

2

Выбор направления

В открывшемся окне выберите направление энергосбережения из справочника, кликнув по строке.

3

Заполнение данных мероприятия

После выбора направления заполните следующие поля:

  • Наименование мероприятия — описание мероприятия
  • Объем внедрения — в соответствующих единицах
  • Условно-годовой экономический эффект, т у.т. — в тоннах условного топлива
  • Условно-годовой экономический эффект, руб. — рассчитывается автоматически на основе текущего курса доллара и стоимости 1 т у.т.
  • Ожидаемый срок внедрения — квартал (1-4)
  • Ожидаемый экономический эффект в текущем году, т у.т.

Поле «Условно-годовой экономический эффект, руб.» рассчитывается автоматически по формуле:

Эффект (руб.) = Эффект (т у.т.) × Стоимость 1 т у.т. (USD) × Курс доллара (BYN)

Курс доллара и стоимость 1 т у.т. указываются на титульном листе плана.

Важное ограничение:

Ожидаемый экономический эффект от внедрения мероприятия в текущем году не может превышать Условно-годовой экономический эффект, т у.т.

Эффект в текущем году ≤ Условно-годовой эффект (т у.т.)

При превышении система отобразит предупреждение и не позволит сохранить мероприятие.

4

Финансовые показатели

Укажите источники финансирования мероприятия и нажмите на "Добавить мероприятие" для сохранения данных.

Для показателей с двойным эффектом (Экономия и Увеличение), финансы не указываются.

5

Итоговые строки

Система автоматически рассчитывает итоговые значения по разделам:

  • Итого по разделу 2.1 — для первоначальных мероприятий
  • Итого по разделу 2.2 — для мероприятий, добавленных при корректировке
  • Всего экономия ТЭР — сумма по разделам 2.1 и 2.2

Раздел 3. Мероприятия по увеличению использования местных ТЭР

В этом разделе отражаются мероприятия, направленные на увеличение использования местных топливно-энергетических ресурсов. Процесс заполнения аналогичен разделу 2, но с одним отличием:

Отличие раздела 3:

Поле «Ожидаемое увеличение использования местных ТЭР в текущем году, т у.т.» пользователь заполняет самостоятельно, без ограничений по сравнению с условно-годовым эффектом.

1

Добавление мероприятия

Нажмите кнопку «Добавить» в таблице мероприятий по увеличению использования местных ТЭР.

2

Выбор направления

В открывшемся окне выберите направление энергосбережения из справочника (коды 1610-1699).

3

Заполнение данных мероприятия

После выбора направления заполните следующие поля:

  • Наименование мероприятия — описание мероприятия
  • Объем внедрения — в соответствующих единицах
  • Условно-годовое увеличение использования местных ТЭР, т у.т. — в тоннах условного топлива
  • Условно-годовое увеличение использования местных ТЭР, руб. — рассчитывается автоматически
  • Ожидаемый срок внедрения — квартал (1-4)
  • Ожидаемое увеличение использования местных ТЭР в текущем году, т у.т. — заполняется пользователем самостоятельно
4

Финансовые показатели

Укажите источники финансирования мероприятия и нажмите на "Добавить мероприятие" для сохранения данных.

5

Итоговые строки

Система автоматически рассчитывает итоговые значения по разделам:

  • Итого по разделу 3.1 — для первоначальных мероприятий
  • Итого по разделу 3.2 — для мероприятий, добавленных при корректировке
  • Всего увеличение использования местных ТЭР — сумма по разделам 3.1 и 3.2

Важно

Все изменения в таблицах фиксируются в системе автоматически после нажатия кнопок «Применить изменения» или «Добавить».

Перед отправкой плана на согласование система проводит автоматический контроль заполнения всех разделов.

Контроль и отправка плана

Перед отправкой плана на согласование система проводит автоматическую проверку заполнения всех разделов. После успешного прохождения контроля план можно отправить на согласование и утверждение.

Контроль заполнения плана

Система автоматически проверяет корректность заполнения всех разделов плана. Контроль осуществляется по следующим направлениям:

📋

Обязательные поля

  • • Период (год)
  • • Вид документа
  • • Наименование организации
  • • Код ОКПО (12 знаков)
  • • Региональное управление
  • • Должность и ФИО руководителя
  • • Контактные данные исполнителя
  • • Курс доллара
📊

Часть 1. Показатели ТЭР

  • ✓ Сумма строк 2-22 = строка 1
  • ✓ Электроэнергия: т у.т. = тыс. кВт·ч × 0,123
  • ✓ Теплоэнергия: т у.т. = Гкал × 0,143
  • ✓ СПТЭР = КПТ + ЭЭ + ТЭ (с учетом знаков)
  • ✓ Экономия ТЭР ≥ заданию на титуле
  • ✓ Доля местных ТЭР ≥ заданию на титуле
  • ✓ Доля ВИЭ ≥ заданию на титуле
📈

Часть 2. Мероприятия по экономии

  • • Сквозная нумерация строк
  • • Код направления из справочника
  • • Эффект (руб.) = Эффект (т у.т.) × стоимость 1 т у.т. × курс
  • • Эффект в текущем году ≤ общему эффекту
  • • Срок окупаемости ≤ 10 лет
  • • Сумма источников = общему финансированию
🌿

Часть 3. Увеличение местных ТЭР

  • • Коды направлений только 1610-1699
  • • Сквозная нумерация строк
  • • Эффект (руб.) = Эффект (т у.т.) × стоимость 1 т у.т. × курс
  • • Сумма источников = общему финансированию

Контроль по нарастающим периодам:

  • январь-декабрь ≥ январь-сентябрь ≥ январь-июнь ≥ январь-март
  • Значения по периодам должны соответствовать итоговым строкам

Контроль плана

На странице с заданиями плана необходимо выбрать Пройти контроль.

Отправка плана на согласование

После успешного прохождения контроля план можно отправить на согласование и утверждение. Процесс отправки включает несколько шагов:

1

Отправить план

На странице с заданиями плана необходимо выбрать Отправить план.

2

Выбор организаций для согласования

В первом шаге выберите организации, которые будут согласовывать план. Можно выбрать несколько организаций.

Автоматический выбор: Первая организация для согласования автоматически определяется как региональное управление по надзору за рациональным использованием ТЭР.

3

Выбор утверждающей организации

На втором шаге выберите организацию, которая будет утверждать план. Можно выбрать только одну организацию.

4

Загрузка сертификата ЭЦП

Для отправки плана необходимо прикрепить файл сертификата электронной цифровой подписи (.cer).

Важно

ПОКА ЧТО ОТКЛЮЧЕНО

Путь согласования

После отправки план проходит следующий путь в зависимости от того какие огранизации были выбраны при отправке плана на проверку.

ℹ️ Примечание:

Ошибки при заполнении плана отображаются в интерфейсе с пояснением, что именно требуется исправить.
